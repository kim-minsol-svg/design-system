## 👋 소개

안녕하세요. 프로덕트 디자이너 김민솔 입니다.

사업 전략을 UX 관점으로 구조화합니다. Claude Code, Cursor 등 AI 도구를 적극 활용해 디자인 프로세스를 최적화하고 있습니다. 
GA·Firebase를 기반으로 문제를 정의하고 개선한 경험이 있으며, Flutter, iOS/AOS, Web 등 다양한 환경의 리소스를 제작하며 개발 조직과 긴밀하게 협업해왔습니다. 
데이터와 AI 도구를 결합한 UX로 프로덕트의 실질적 가치를 높이는 디자이너를 지향합니다.
